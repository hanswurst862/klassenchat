Ausführliche Beschreibung

Dieser Dialog ist nur für den Chat::oberadmin.

Dieser Dialog zeigt alle Nutzer mit ihrem Benutzername_str an und (falls sie einer der Chat::std_admins sind) mit ihrem Passwort. Sie können dann zum Admin gemacht werden oder entmachtet werden.
