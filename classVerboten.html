Ausführliche Beschreibung

Dieser Dialog zeigt alle verbotenen Benutzernamen an.

Es können Benutzernamen hinzugefügt werden, indem das itemneu bearbeitet wird. Es können Benutzernamen entfernt werden, indem der Text gelöscht wird. Es können Benutzernamen verändert werden, indem der Text bearbeitet wird.
